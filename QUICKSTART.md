# Quick Start Guide

## Installation & Run

1. **Install dependencies:**
```bash
cd cellular_network_dashboard
npm install
```

2. **Run the application:**
```bash
./run.sh
```

Or manually:
- Terminal 1: `npm run server` (starts data API on port 3001)
- Terminal 2: `npm run dev` (starts React app on port 3000)

3. **Open browser:**
Navigate to http://localhost:3000

## What's Different from Streamlit Version?

✅ **React-based** - Modern, fast, professional UI
✅ **No static images** - All visualizations generated dynamically from data
✅ **Better performance** - Client-side rendering with React
✅ **Modern UI** - Tailwind CSS with professional design
✅ **Interactive charts** - Recharts with zoom, pan, hover
✅ **Type-safe** - TypeScript for better code quality

## Features

- **Overview Page**: Key statistics and project summary
- **User Analysis**: Individual user data with dynamic charts
- **Tower Recommendations**: Interactive maps and priority analysis
- **Visualizations**: Comprehensive dynamic charts from data

All charts are generated on-the-fly from your CSV/JSON data files - no pre-rendered images!
